# Olist E-commerce Data Analysis (SQL Project)

## 📊 Project Overview

This project analyzes the Brazilian e-commerce dataset from Olist to understand customer purchasing behavior, sales performance, and product trends.

The goal of this analysis is to extract meaningful business insights using SQL queries and data analysis techniques.

## 🎯 Objectives

* Analyze overall sales performance
* Identify top performing product categories
* Understand customer purchase behavior
* Perform customer segmentation using RFM analysis
* Study order trends over time

## 🛠 Tools & Technologies

* SQL (MySQL)
* Data Analysis
* Relational Database Queries

## 📂 Dataset

The dataset contains information about:

* Customers
* Orders
* Order items
* Payments
* Products
* Sellers
* Product category translation

## 📈 Key Analysis Performed

### 1️⃣ Revenue Analysis

* Total revenue generated
* Top product categories by revenue

### 2️⃣ Customer Analysis

* Unique customers
* Repeat vs new customers

### 3️⃣ Order Trend Analysis

* Monthly order growth
* Seasonal purchasing patterns

### 4️⃣ Customer Segmentation (RFM)

Customers were segmented using:

* Recency
* Frequency
* Monetary value

This helps identify:

* Loyal customers
* Potential customers
* At-risk customers

## 📊 Key Insights

* Certain product categories contribute the majority of revenue.
* A small percentage of customers generate a large share of sales.
* Seasonal spikes are visible in monthly order trends.
